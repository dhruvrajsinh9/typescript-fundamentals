## Learning Rules

### Kaizen Approach

Improve by 1% every day.

Daily focus:

- Learn one concept
- Write code
- Commit progress
- Review previous concepts

### 80/20 Rule

- 20% Theory
- 80% Coding

### Active Learning

Instead of only watching tutorials:

1. Learn a concept
2. Implement examples
3. Modify examples
4. Build a small feature
5. Commit changes
